# OpenClaw Workspace

Welcome to the OpenClaw workspace! This is the central hub for all OpenClaw operations, including AI assistants, automation scripts, and business tools.

## Workspace Organization

This workspace has been organized into a structured directory system for better management. See `README_ORGANIZATION.md` for detailed information about the folder structure.

## Key Components

- **Daily AI News**: Automated system delivering AI news every day at 9 AM CT
- **Monitoring Systems**: Various scripts to monitor system status and operations
- **Communication Tools**: Telegram integration and messaging systems
- **Business Tools**: Various utilities for business operations

## Getting Started

1. **Explore the directory structure** to understand where different components are located
2. **Check the documentation** in the `docs/` directory for detailed guides
3. **Review the scripts** in the `scripts/` directory for automation tools
4. **Monitor ongoing operations** using the tools in the `tools/` directory

## Directories

- `agents/`: Agent-related files and configurations
- `config/`: Configuration files
- `docs/`: Documentation and guides
- `logs/`: Log files
- `memory/`: Memory and history files
- `scripts/`: Automation scripts
- `tools/`: Utility tools
- `web/`: Web interface files
- `projects/`: Project-specific files
