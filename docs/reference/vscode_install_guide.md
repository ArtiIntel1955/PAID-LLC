# Visual Studio Code Installation Guide

## Manual Installation Steps

1. Open your web browser
2. Go to https://code.visualstudio.com/
3. Click on the "Download for Windows" button
4. Once downloaded, run the installer (VSCodeUserSetup-{version}.exe)
5. Follow the installation wizard:
   - Accept the license agreement
   - Choose the installation location (default is recommended)
   - Select "Add to PATH" if prompted
   - Complete the installation

## Why VS Code is Important for Your Business Website

- **Free and Open Source**: Completely free for commercial use
- **Powerful Features**: Syntax highlighting, debugging, Git integration
- **Extensive Extensions**: Thousands of free extensions for web development
- **Live Preview**: See changes instantly with extensions
- **Integrated Terminal**: Run commands without leaving the editor

## Recommended Extensions for Web Development
After installing VS Code, consider adding these free extensions:
- Live Server (to preview your website locally)
- Prettier (to format your code)
- Auto Rename Tag (for HTML editing)
- GitLens (enhanced Git capabilities)
