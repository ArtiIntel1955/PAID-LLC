# Business Website Development Setup

## Free Tools Needed

### 1. Code Editor
- **Visual Studio Code (VS Code)** - Download from https://code.visualstudio.com/
  - Lightweight and powerful
  - Excellent extensions for web development
  - Built-in terminal and Git integration

### 2. Version Control
- **Git** - Download from https://git-scm.com/
  - Essential for tracking changes
  - Required for GitHub integration

### 3. Web Hosting
- **GitHub** - Sign up at https://github.com/
  - Create a repository named `username.github.io`
  - Enable GitHub Pages in repository settings
  - Free hosting for static websites

### 4. Design Tools
- **Figma** - Free account at https://figma.com/
  - Design your website mockups
  - Collaborate on designs

### 5. Image Resources
- **Unsplash** (https://unsplash.com/) - High-quality free photos
- **Pexels** (https://pexels.com/) - Additional free photos
- **Feather Icons** (https://feathericons.com/) - Simple, clean icons

### 6. Development Languages
- **HTML5** - Structure
- **CSS3** - Styling
- **JavaScript** - Interactivity
- **Responsive Design** - Mobile-friendly layouts

## Getting Started Steps

1. Install VS Code
2. Install Git
3. Create GitHub account
4. Create your first HTML file
5. Push to GitHub and enable GitHub Pages

## Sample Tech Stack for Business Website
- HTML5 for structure
- CSS3 with Flexbox/Grid for layout
- JavaScript for interactivity
- Responsive design for mobile
- GitHub Pages for hosting
- Custom domain when ready
